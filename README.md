# Resume
  Resume built using [R pagedown](https://github.com/rstudio/pagedown) package.

## Build Instructions

```shell
$ docker pull ragumanjegowda/docker:latest
$ docker run -v $PWD:/mnt -it ragumanjegowda/docker:latest /bin/bash
$ cd /mnt
$ ./build.sh
```
